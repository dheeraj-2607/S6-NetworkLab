# CSL322 Networking Lab

This repository contains the programs and experiments done as part of the **CSL322 Computer Networks Lab (2024–25)** at **Government Engineering College Idukki**.

## Experiments

1. TCP echo server and UDP echo server
2. TCP client-server program (string reverse)
3. UDP client-server program (addition of two numbers)
4. Implementation of FTP `get` and `put` commands
5. Link State Routing algorithm
6. Stop and Wait flow control
7. Go-Back-N ARQ protocol
8. TCP 3-way handshake using Wireshark😥
9. Multi-client UDP server using epoll😥
10. Study of NS-2 network simulator😥
